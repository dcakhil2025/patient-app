Patient App - Integrated version
How to use:
1. Unzip the folder.
2. Open index.html in your browser to use the dashboard.
3. Add a patient, open profile, and click 'Open Investigations Table' to view the vertical investigations table.
Files included:
- index.html, add.html, patient.html (main app)
- /assets/ (logo, pavithra.docx, Investigations Chart.pdf if uploaded)
- /investigations/index.html (vertical investigations table)
